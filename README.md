# xiaohao-toolkit

项目公共工具类库

## 使用方法

- groovy

```groovy
implementation platform("com.xiaohaoo.dependencies:xiaohao-dependencies:1.0.0")
implementation "com.xiaohaoo:xiaohao-toolkit"
```

- kotlin

```kotlin
implementation(platform("com.xiaohaoo.dependencies:xiaohao-dependencies:1.0.0"))
implementation("com.xiaohaoo.toolkit:xiaohao-toolkit")
```
